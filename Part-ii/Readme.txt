Part -ii
----------
The script in this part gets list of customers as in part-i. However, in this part few changes have been made for pagination in database.
When data is huge, we might need to query only certain records at a time. 

Changes in scripts
-Additional Parameters
1. @PageNumber : Passed as input parameter for the page number.
2. @PageSize   : Passed as input parameter for size of page or no of rows required from records.
